# Chimera - The Wordpress Connection Plugin
### /kīˈmirə/
> A mythical animal with parts taken from various animals.
## Overview
This plugin is the bridge between tigongolfcarts.com and the DMS. It combines these two very distinct beast into one cohesive unit, not unlike creating a chimera.
Chimera is responsible for creating and maintaining Woocommerce product pages for cart inventory, and allowing the DMS to control certain Elementor elements.
## Requirements
- WooCommerce
- Elementor
## Addons
- PDF Embedder
- Custom Product Tabs for WooCommerce
- Yoast SEO
- Woocommerce Custom Product Addons
- Google for WooCommerce
- Facebook for WooCommerce
- Pinterest for WooCommerce
- Tigon Watermark Plugin
